Notebook sharing

  • As the pywedge uses the widgets heavily to display interactive UI, sharing the notebook in static webpages like GitHub, may not render the widgets. It is observed that widgets are not displaying properly in shared kaggle notebook.
  • It is easy to share the notebook using Google Colab, it automatically saves the widget state.
  • In local machine Jupyter notebook, the widget state can be saved from the widgets tab.
